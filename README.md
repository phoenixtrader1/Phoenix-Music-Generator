# Phoenix-Music-Generator
Phoenix Music Generator is a web-based platform where users spend tokens to generate custom music tracks. Designed for simplicity and Web3 utility, it includes a token system, user wallets, and a clean web dashboard. Early supporters can use the platform and explore token-gated AI music generation.  
# Phoenix Music Generator

Phoenix Music Generator is a token-gated web platform where users can generate custom music tracks.

## Features
- Token-based access
- Web-based dashboard
- Simple, clean interface
- Early supporter perks can be added

## Folder Structure
phoenix-music-generator/
├── ai/        # Music generation logic
├── users/     # Wallet and token balances
├── utils/     # Token spend utilities
├── web/       # Templates and static files
├── config.py
├── main.py
├── requirements.txt
├── README.md
└── .env.example

## Usage
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `python main.py`
4. Open browser: `http://127.0.0.1:5000/`
5. Access dashboard with `?user=user1` or `?user=user2`
